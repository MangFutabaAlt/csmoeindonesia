# CSMoE Language Patcher

A simple one-click patcher to install and manage language translations for CSMoE.

This tool automatically downloads and applies the latest translation patch directly from the repository, making the process easier for players who want to play CSMoE in their preferred language.

---

# Features

- One-click language patch installation
- Automatic update check from GitHub
- Multi-language support
- Switch languages anytime
- No manual file replacement required

---

# Supported Languages

- English
- Indonesian
- Vietnamese (coming soon)

More languages may be added in the future with community contributions.

---

How It Works?

The launcher will:

1. Detect your language CSMoE 
2. Check the latest translation version from the GitHub repository
3. Download the required patch files
4. Apply the selected language automatically

You can change languages anytime through the patcher.

---

Installation

1. Download the latest patcher executable.
2. Run the launcher
3. Select the directory that contains the csmoe folder. 
4. Select your language and apply the patch.

That's it, no hard installation needed.

---

No additional runtime or dependencies are required.
